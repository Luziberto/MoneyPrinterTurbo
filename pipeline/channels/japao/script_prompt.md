Crie um roteiro para Facebook Reels, Instagram Reels, YouTube Shorts e TikTok.

Regras:

- Duração entre 60 e 90 segundos.
- Entre 120 e 180 palavras.
- Público brasileiro.
- Explique apenas UMA curiosidade sobre {niche}.
- O roteiro deve parecer uma pequena história.
- Comece com uma pergunta ou afirmação intrigante.
- Desperte curiosidade nos primeiros 2 segundos.
- Não revele a resposta completa na primeira frase.
- Use linguagem simples e natural.
- Use frases curtas.
- Cada frase deve ter no máximo 12 palavras.
- Evite frases longas com muitas vírgulas.
- Cada frase deve funcionar bem como legenda.
- Cada parágrafo deve aumentar a curiosidade do espectador.
- Inclua contexto cultural para ajudar brasileiros a entenderem o assunto.
- Mantenha uma pergunta implícita durante todo o roteiro.
- Crie expectativa entre os parágrafos.
- Não use listas.
- Não use tópicos.
- Não use numeração.
- Não use emojis.
- Não escreva título.
- Não escreva observações.
- Não escreva marcações.
- Não use chamadas para ação como "curta", "comente", "compartilhe" ou "siga".
- O último parágrafo deve terminar com uma informação surpreendente, uma reflexão ou uma curiosidade que deixe o espectador querendo descobrir mais.
- O final deve transmitir sensação de continuidade natural, sem parecer um anúncio.
- Evite repetir assuntos já abordados anteriormente.
- O objetivo é maximizar retenção, compartilhamentos e tempo de exibição.

Importante:

- Retorne apenas o roteiro final.
